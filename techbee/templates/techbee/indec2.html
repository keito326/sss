Parts posting

パーツ投稿

会員様一人一人がオリジナルのデザインパーツを投稿し、互いに評価しあう事で、スキルに磨きをかける場を用意致しました。それに加えて、評価を受けることで現金にも交換できるポイントが付与されるシステムを取り入れております。また、投稿されたパーツはご自由にお使いいただけます。

Regular gatherings

定期的な交流会

月に一度、会員様同士が交流できる会を開催いたします。学習を進めていく上で、モチベーション管理も重要なポイントです。同じ目標を持つ仲間同士で集まり、意見交換をしたり、息抜きをしたりなど、各々有効に利用していただけたらと思います。

study support

学習サポート

現役で働いているエンジニアが学習のサポートをさせていただきます。学習を始める上での環境構築や、進め方、問題解決の方法、ノウハウなど、自走するための手助けをさせていただきます。そのため未経験の方でも着実にスキルが身につく環境となっております。

想像を、創造する。

 さぁ、プログラミングを始めよう。

HTML

HTMLとは、Hyper Text Markup Language（ハイパーテキスト・マークアップ・ランゲージ）の略で、Webページを作るための最も基本的なマークアップ言語のひとつです。普段、私たちがブラウザで観ているWebページのほとんどが、HTMLで作られています。

CSS

CSSとは、“Cascading Style Sheets” の頭文字をとったものであり、スタイルシートとも呼ばれます。HTMLが、見出しやヘッダ情報などWebページの文書構造を形作るための言語であるのに対し、CSS言語は文書構造にデザインを施し、見栄えを整える役割を担っています。

jQuery

素敵な動きを手軽に実装できるJavaScriptライブラリです。 アニメーション効果をつけたり、Ajax（エイジャックス）を使って外部ファイルを読み込んだりと色々なことができます。

Python

Pythonとは、組み込み開発、WEBアプリケーション、デスクトップアプリケーションなどで利用される言語のことです。
言語としての特徴では、文法がとても単純で、だれが書いても同じような感じになり可読性が高いです。
